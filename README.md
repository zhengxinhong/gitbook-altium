## （二）快捷键


* 测距：单击R+M或Ctrl+M光标出现十字即可进行测距
* 单位转换：使用快捷键Q(注意是英文状态下)。
* 查看元件：右击--> 查看 --> 适合所有对象 / ctrl+PgDn
* 2D与3D空间的转化2D -> 3D ：按数字3键
* 3D -> 2D：按数字2键。
* PCB拖动元器件：E --> M -- >D
